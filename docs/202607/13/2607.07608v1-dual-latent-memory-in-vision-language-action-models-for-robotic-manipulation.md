## 摘要
主流的视觉-语言-动作（VLA）模型在马尔可夫假设下主要根据当前观测预测动作，因此在处理长时域、时间依赖的任务时表现不佳。现有的记忆增强型VLA要么扩展观测窗口，要么从记忆库中检索历史信息作为辅助策略端上下文。然而，它们将记忆排除在VLA推理的原生潜在嵌入空间之外，阻止了历史经验与多模态推理和动作生成的流畅交织。为此，我们提出了LaMem-VLA，一种原生潜在记忆框架，它将历史经验重构为潜在记忆标记，并直接将其与VLA推理交织在一起。其核心引入了四个协调组件：（i）策展器，将历史经验组织成互补的短期和长期记忆库；（ii）搜索器，利用多模态认知查询两个记忆库以检索上下文相关证据；（iii）压缩器，将检索到的证据重构为紧凑的短期和长期潜在记忆标记；（iv）编织器，将这些记忆标记与当前观测和指令注入到一个连续的嵌入序列中。通过在相同的连续潜在空间中完全表示、检索和消费历史经验，LaMem-VLA使记忆能够直接参与VLA推理，并在有限上下文中指导动作生成。在SimplerEnv和LIBERO上的大量实验证明了LaMem-VLA的优越性。

## Abstract
Mainstream Vision-Language-Action (VLA) models predict actions primarily from the current observation under a Markovian assumption, thus struggling with long-horizon, temporally dependent tasks. Existing memory-augmented VLAs either expand the observation window or retrieve history from the memory bank as auxiliary policy-side context. However, they leave memory outside the native latent embedding space of VLA reasoning, preventing historical experience from being fluidly interleaved with multimodal reasoning and action formation. To this end, we introduce LaMem-VLA, a latent-memory-native framework that reconstructs historical experience into latent memory tokens and directly interweaves them with VLA reasoning. At its core, LaMem-VLA introduces four coordinated components: (i) a curator that organizes historical experience into two complementary short-term and long-term memory vaults; (ii) a seeker that queries both vaults using the multimodal cognition to retrieve context-relevant evidence; (iii) a condenser that reconstructs the retrieved evidence into compact short-term and long-term latent memory tokens; and (iv) a weaver that injects these memory tokens with the current observation and instruction into one continuous embedding sequence. By representing, retrieving, and consuming historical experience entirely in the same continuous latent space, LaMem-VLA enables memory to directly participate in VLA reasoning and guide action generation under a bounded context. Extensive experiments on SimplerEnv and LIBERO demonstrate the superiority of our LaMem-VLA.

- **问题**：主流 VLA 模型在马尔可夫假设下仅利用当前观测预测动作，缺乏对历史信息的显式利用，导致在长时域、时间依赖的操纵任务中表现不佳（“时间短视偏差”）。
- **现有方法的局限**：①扩展现有观测窗口的方法计算成本随长度增长，且固定窗口会丢弃窗口外的信息；②外部记忆库方法将历史信息存储在 VLA 模型的原生 token 空间之外，仅作为辅助策略端上下文，无法与多模态推理和动作形成过程深度融合。
- **论文目标**：将历史经验表示为**上下文原生的潜在记忆**，在相同的连续潜在空间中完成存储、检索和消费，使记忆能直接参与 VLA 的推理与动作生成。

### 2. 方法论
- **核心思想**：构建双尺度（短期视觉、长期语义）潜在记忆库，通过四个协调组件将历史经验重构为紧凑的潜在记忆 token，并直接嵌入到 VLA 的输入序列中，与当前观测、指令和动作查询一起进行自注意力计算。
- **关键组件**：
  - **记忆策展器（Curator）**：将历史经验组织成两个互补的记忆库。短期记忆库存储当前 episode 的视觉 token（经压缩模块 SE-bottleneck 处理），长期记忆库存储动作 hidden state 以跟踪任务进度和动作连续性。当单元数超过容量 L（=16）时，使用余弦相似度合并最冗余的相邻对。
  - **记忆搜寻器（Seeker）**：基于当前多模态认知（视觉 token + 指令 token）构建上下文感知查询 Q_cont，并附加可学习查询槽 Q_init，通过轻量级 Transformer 查询构建器 B 生成共享查询 Qt。利用余弦相似度从短期库和长期库各检索 Top‑K（=8）个单元。
  - **记忆压缩器（Condenser）**：将检索到的原始证据（Z_short、Z_long）与上下文查询 Qt 和可学习记忆槽（Ts、Tl）一起输入轻量级记忆形成器（Fv、Fc），输出固定长度的短时记忆 token M_short（Ls=8）和长时记忆 token M_long（Ll=4）。
  - **记忆编织器（Weaver）**：将压缩后的记忆 token 加上可学习源嵌入后，与当前视觉 token Xt、指令 token I 和动作查询 Q_action 拼接成记忆增强的 VLA 输入序列 St，使记忆 token 在自注意力中直接参与推理。

- **动作生成**：记忆增强后的动作 token Z_action 作为条件，输入扩散策略专家（约300M参数），通过 DDIM 采样10步生成连续7‑DoF 动作块（horizon=16）。训练用 MSE 损失预测添加的噪声。

### 6. 主要结论与发现
- LaMem-VLA 在 SimplerEnv‑Bridge 上平均成功率 73.9%，超越 π0（69.2%）和 MemoryVLA（71.9%）；在 LIBERO 上平均 97.6%，超越 MemoryVLA（96.5%）和 CogACT（93.2%）。
- **双记忆互补**：短期视觉 token 保留细粒度感知证据，长期语义 token 跟踪任务进度和动作连续性，两者缺一不可。
- **潜在原生融合更优**：将记忆 token 直接嵌入 VLA 输入序列进行自注意力，优于仅作为外部条件或原始检索拼接。
- **检索量平衡**：K=8 获得最佳性能，太少证据不足，太多引入冗余。
- **记忆 token 数**：需要平衡性能与计算开销，默认 Ls=8、Ll=4 效果较好。

### 7. 优点
- **范式创新**：首次将机器人历史记忆视为 VLA 模型原生潜在上下文的一部分，实现记忆与感知、推理、动作生成的深度耦合。
- **双尺度设计**：显式区分短期视觉记忆和长期语义记忆，互补覆盖不同时间跨度的需求。
- **高效紧凑**：压缩器将检索到的可变长度证据重构为固定少量 token，避免上下文膨胀。
- **实验全面**：在两个标准 bench 上对比大量基线，消融实验覆盖设计关键，分析深入。

### 8. 不足与局限
- **仅限仿真**：论文明确指出当前版本仅在模拟环境中验证，真实世界实验（如机器人平台）仍在进行中，泛化性和实际鲁棒性尚未证明。
- **资源消耗**：7B骨干 + 扩散专家 + 记忆模块需要较高计算资源（8 × H800），部署成本高。
- **记忆管理策略简单**：记忆压缩（相邻合并）和检索（余弦相似度+Top‑K）较为基础，可能对长轨迹中复杂依赖的建模不够精细。
- **超参数固定**：记忆容量 L、检索数 K、记忆 token 数等在实验中固定，未展示针对不同任务的自适应调节能力。
- **未讨论多模态对齐与遗忘机制**：长期记忆的语义漂移或错误累积未被显式处理。
