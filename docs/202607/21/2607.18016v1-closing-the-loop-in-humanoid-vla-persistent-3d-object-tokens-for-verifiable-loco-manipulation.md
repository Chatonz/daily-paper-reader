## 摘要
视觉-语言-动作策略是通用机器人控制的有前途的基础，但长期的人形机器人移动操作要求机器人将任务对象视为在运动、接触、遮挡和恢复过程中持续的物理实体。我们将此问题研究为对象状态分歧：用于调节全身动作的对象状态可能与用于判断动作是否达到预期物理关系的状态不同。我们提出**持久对象令牌化**（POT），该方法从RGB-D观测中维护角色索引的3D对象记录，并将其转换为用于全身动作专家的对象令牌。实例化为**POT-VLA**，相同的对象记录同时调节动作生成并支持几何谓词检查，产生一个闭环执行系统，其中对象状态既可操作又可验证。在Unitree G1上，POT-VLA在8个真实世界任务族中将匹配的直接GR00T-N1.7基准从39/80成功提升至71/80。在与外部Being-0对齐的参考中，POT-VLA在对齐服务任务上达到44/50成功，而Being-0论文报告为37/50。最大的改进出现在需要维持3D关系的任务上，表明持久对象中心状态是可验证类人VLA执行的有用抽象。

## Abstract
Vision-language-action policies are a promising foundation for general robot control, but long-horizon humanoid loco-manipulation requires the robot to treat task objects as persistent physical entities across movement, contact, occlusion, and recovery. We study this problem as object-state divergence: the object state used to condition a whole-body action can differ from the state used to decide whether the action achieved the intended physical relation. We propose \emph{Persistent Object Tokenization} (POT), which maintains role-indexed 3D object records from RGB-D observations and converts them into object tokens for a whole-body action expert. Instantiated as \emph{POT-VLA}, the same object records condition action generation and support geometric predicate checks, yielding a closed-loop execution system in which object state is both actionable and verifiable. On a Unitree G1, POT-VLA improves a matched direct GR00T-N1.7 baseline from 39/80 to 71/80 successes over eight real-world task families. In an external Being-0-aligned reference, POT-VLA achieves 44/50 successes on aligned service tasks, compared with the 37/50 success reported by the Being-0 paper. The largest gains occur on tasks requiring maintained 3D relations, suggesting that persistent object-centered state is a useful abstraction for verifiable humanoid VLA execution.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：长期（long-horizon）人形机器人移动操作（loco-manipulation）面临的核心困难是：任务对象（如杯子、篮子、支撑面等）必须作为持久的物理实体被机器人持续追踪，即使经历机器人行走、抓取、放置、接触、遮挡和恢复等过程。当前主流视觉-语言-动作（VLA）策略通常基于当前观测的视觉-语言特征来生成动作，而任务进展的验证可能由独立的监视器或语言级状态进行，导致**对象状态分歧**——用于条件化动作生成的对象状态与用于验证动作是否达到预期物理关系的对象状态可能不同。这种不一致会引发抓取不准确、放置偏移、错误的对象操作、无效恢复等问题。

- **整体含义**：论文提出**持久对象令牌化**（POT）方法，旨在将对象中心的状态同时用于动作生成和物理关系验证，从而形成一个**闭环可验证**的人形VLA执行系统。通过共享的对象记忆（object memory）消除状态分歧，提升长时任务的成功率。

## 2. 方法论：核心思想、关键技术细节、流程

### 核心思想
- 提出**持久对象令牌化（POT）**，从RGB-D观测中维护**角色索引的3D对象记录**（role-indexed 3D object records），并将这些记录序列化为**持久3D对象令牌**，供全身动作专家（whole-body action expert）使用。
- 共享的对象记忆在每次动作块执行后通过新的RGB-D观测刷新，然后由**几何谓词验证器**检查是否达到预期的物理关系（如是否在容器内、是否支撑稳定等），从而形成“维护-行动-观测-验证-恢复”的闭环。

### 关键技术细节
1. **任务角色对象记录**：给定指令和当前观测，由VLM规划器或任务文件提供类型化子任务计划。每个子任务定义实体集，POT构建角色索引的对象记忆，每条记录存储实体角色、短语、2D框、3D质心和范围、置信度、可见性、关系特征等。使用SAM3获取角色/查询掩码，有效深度像素通过相机内参和相机到基座的变换投影，在机器人基座坐标系下汇总。
2. **持久令牌模式**：将对象记忆映射为固定槽（K=8）的令牌张量，每槽33个特征（包括可见性、置信度、归一化2D框、3D质心、3D范围、可选方向、关系特征）。槽位由角色ID标识（如TARGET, DESTINATION, SUPPORT, HANDOVER_PARTNER），未用槽为PADDING并掩码。
3. **对象令牌条件VLA执行**：以GR00T-N1.7作为动作专家，POT-VLA在动作头序列中插入对象令牌。具体地，在DiT动作头的自注意力序列中，布局为[状态令牌, 对象令牌, 动作令牌]，视觉-语言骨干特征作为交叉注意力的隐藏状态。对象令牌通过一个投影器（LayerNorm→Linear→GELU→Dropout→Linear）映射到DiT隐藏维度，并添加可学习的角色嵌入和上下文嵌入。训练时，演示数据配对对象令牌侧车，使用相同的动作损失，更新对象令牌分支和动作头参数，不添加辅助损失。
4. **谓词级验证与恢复**：几何谓词监督器读取与动作策略相同的角色索引对象记忆，检查子任务就绪和完成。谓词类型包括containment, support, end-effector–target proximity, target–destination displacement等。每个谓词定义比较运算符、阈值和稳定窗口。验证结果返回in_progress, done, blocked, failed, uncertain。对low-confidence或不可见对象触发重新观测或重新接地。基于诊断结果选择继续、重试、重新观测、重新接地或重新规划。

## 6. 主要结论与发现

- **POT-VLA显著提升了任务成功率**：在80次总体试验中从39/80（48.75%）提升至71/80（88.75%）。在外部Being-0对齐任务上达到44/50（88%），超过Being-0报告的37/50（74%）。

## 7. 优点

- **问题定义清晰**：明确提出“对象状态分歧”这一核心问题，切中长时人形操作的痛点。
- **方法设计巧妙**：POT将持久对象记忆与VLA策略深度融合，既作为动作条件又作为验证依据，形成闭环；不依赖模拟器、物理引擎或学习动力学模型，仅基于RGB-D观测。
- **复用现有模型**：动作专家基于GR00T-N1.7，通过对象令牌增强而非重新训练整体策略，易于集成。
- **验证机制实用**：几何谓词监督器使用可解释的度量标准，支持细致诊断和局部恢复，避免了完全依赖语言级反馈的不确定性。
- **实验设计严谨**：控制变量对比、消融、泛化测试，对比基线和外部工作，结果支撑结论。

## 8. 不足与局限

- **依赖感知质量**：POT依赖SAM3和RGB-D传感器的可靠性，以及相机标定精度。视觉遮挡、低纹理、光照变化等可能降低对象记录质量，进而影响整个系统。
